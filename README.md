# Starter Carpentry Website

Built with Next.js. Static export.

## Requirements

Node.js 18+

## Commands

```bash
npm install   # install dependencies
npm run generate  # compile briefs → data/ + globals.css
npm run build # build static export to /out
npm run dev   # local preview at localhost:3000
```

## Deploy

Deploy the contents of `/out` to any static host.

- **Vercel**: Connect this repo, set output directory to `out`
- **Netlify**: Connect this repo, set publish directory to `out`

## Changes

For content updates, new services, or additional suburb pages, contact your developer.
